News

June 26, 2014

Server Restarts: please remember that the PMDS-Data Server will restart every Tuesday from 11:30 AM to 11:40 AM. During this time, you will not be able to save or edit entries in any applications run from the server (SWI Numbers, Shift Reports, etc). During this time, please leave your screen alone and you can continue where you have left off after the restart is complete.
